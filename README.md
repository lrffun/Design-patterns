- 第一次作业（使用和不使用单一职责原则的框架实现Login）
> [使用单一职责原则](https://github.com/lrffun/Design-patterns/tree/master/Login/src/Use_responsibility)

> 总结：使用单一职责原则可以让类更好的被复用，降低类与类之间的耦合，降低每一个类的工作任务，使代码可读性更佳。
